Nesse projeto foram utilizadas as linguagens HTML e CSS no frontend e PHP no backend.
No arquivo HTML foi feito um formulário simples, onde o usuário deverá inserir um número inteiro positivo.
O arquivo CSS é utilizado apenas como complemento estético e funcional ao arquivo HTML, tornando o sistema mais agradável para uso.
O número inserido será enviado para o arquivo PHP, que tem basicamente duas funções: 
	-Inicialmente classifica o número inserido como primo ou não primo; 
	-Logo após, apresenta todos os possíveis divisores do número inserido;
Após a realização dos cálculos, os resultados são exibidos na tela para o usuário.
